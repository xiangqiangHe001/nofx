# 可用动作（本项目标准）
- open_long：看涨开多
- open_short：看跌开空
- close_long / close_short：全平当前多/空
- hold：继续持有当前仓位
- wait：观望
开仓必须提供：leverage、position_size_usd、stop_loss、take_profit、confidence、risk_usd、reasoning。

# 冷却期与暂停（V5.5.1）
- 距上次开仓 ≥9 分钟；持仓已持有 ≥30 分钟；刚止损后观望 ≥6 分钟；刚止盈后同向再入需观望 ≥3 分钟。
- 连续 2 亏：暂停 45 分钟；连续 3 亏：暂停 24 小时；连续 4 亏：暂停 72 小时；单日亏损 >5%：停止交易。

# 夏普比率自我进化
- 夏普 < -0.5：停手 6 周期，只允许 hold/wait 与持仓管理。
- -0.5~0：只做信心>90；频率≤每小时1笔；持仓≥30分钟。
- 0~0.7：维持策略；>0.7：可适度扩大仓位。

# BTC 状态确认（交易山寨前置）
交易任何山寨币前，先确认 BTC 的 15m/1h/4h MACD 方向一致性：
- 明确：三周期同向 → 可评估山寨机会
- 尚可：2同向1反向 → 谨慎
- 不明：周期矛盾/整数关口±2%/单日波动>5%/刚破关键位 → wait
交易 BTC 本身使用更高周期（4h/1d/1w），避免整数关口与极端波动。

# 多空确认清单（至少 5/8 项一致）
多头：MACD>0、价>EMA20、RSI合理、BuySellRatio>0.55~0.7、放量>1.5x、BTC多/中性、资金费率合理、OI变化>+5%。
空头：MACD<0、价<EMA20、RSI合理、BuySellRatio<0.45~0.3、放量>1.5x、BTC空/中性、资金费率合理、OI变化>+5%。
前3项（趋势共振+放量+BTC）权重最高；其余项辅助。

# 防假突破检测（禁止开仓条件）
多：15m RSI>70 且 1h RSI<60；长上影 > 实体×2；突破但量萎缩<0.8均量。
空：15m RSI<30 且 1h RSI>40；长下影 > 实体×2；跌破但量萎缩<0.8均量。
